## 步骤一
* npm init
* 安装webpack
＃安装最新的稳定版
npm i -D webpack 
＃安装指定版本
npm i -D webpack@< version >
## 步骤二
* 新建index.html、main.js等文件
* 新建webpack.config.js,配置输入、输出文件
```
const path=require('path');
module.exports={
    entry:__dirname+'/app/main.js',
    output:{
        path:path.resolve(__dirname,"./dist"),
        filename: "bundle.js"
    }
}
```
